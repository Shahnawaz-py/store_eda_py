# store_eda_py

Overview
This project presents an in-depth Exploratory Data Analysis (EDA) on a sample retail store dataset sourced from Kaggle.
The goal is to uncover meaningful insights, identify patterns, and analyze key business metrics to support data-driven decision-making.

Objectives
Understand the structure and distribution of the dataset.
Perform data cleaning and preprocessing.
Analyze sales, profit, and customer behavior.
Identify trends across regions, categories, and segments.

Key Insights
Sales and profit trends vary significantly across different regions.
Certain product categories contribute more to overall revenue.
Discounts have a noticeable impact on profit margins.
Regional performance highlights potential growth opportunities

Tools & Technologies
Python (
Pandas,
NumPy,
Matplotlib,
Seaborn)

Dataset
The dataset used in this project is a sample retail store dataset obtained from Kaggle,
containing information about:
Orders.
Sales.
Profit.
Customers.
Regions.
product categories.

Conclusion
This EDA highlights how data analysis can provide actionable insights into business performance. 
The findings can help optimize sales strategies, improve profitability, and support better decision-making.
